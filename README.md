# cgi-van
Interactive van done for CGI class

## Commands
  * awsd - move
  * ik - move sattelite
  * jl - rotate sattelite
  * 0 - default view
  * 1 - top view
  * 2 - side view
  * 3 - front view
  * space - change colors
  * f - fill with color / wire mesh
